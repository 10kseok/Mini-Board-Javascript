# Mini Board
React와 Nest로 구성된 간단한 게시판 프로젝트

## 요구사항 
- [x] 회원가입/로그인 (JWT 인증)
- [x] 게시물 작성
- [x] 게시물 목록보기
- [x] 게시물 읽기
- [x] 댓글 작성  
++++++++++++++++++++++++
- [x] 게시물 삭제
- [x] 게시물 수정

## 구성
- Frontend : React.js
- Backend : Nest.js, MySQL
